# kanso-bin

## Installation

```bash
# Using an AUR helper
yay -S kanso-bin
paru -S kanso-bin

# Or manually
git clone https://aur.archlinux.org/kanso-bin.git
cd kanso-bin
makepkg -si
```

## Description

Kanso is a modern media player with forum-style design, focused on convenience, beauty, and performance.

Features:
- Zen-like desktop player with forum-style organization
- Smooth motion and textured surfaces
- Local-first library management
- Video playback with resume logic and timeline preview
